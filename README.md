# Esp32-Oled-Interfacing
This project demonstrates interfacing an SSD1306 128x64 OLED display with an ESP32 using I2C communication.

## Hardware Used
- ESP32 Dev Board
- SSD1306 OLED Display (128x64)
- USB Cable

## Connections
| OLED -- ESP32   |
| SDA  -- GPIO 23 |
| SCL  -- GPIO 21 |
| VCC  -- 3.3V    |
| GND  -- GND     |

## Feature
- Display welcome message
- Uses I2C protocol

## Learning Outcome
- Understanding I2C communications
- OLED display interfacing
- Embedded programming with ESP32
